# 📚 Open Library Book Finder

## 🧾 Project Overview

Open Library Book Finder is a web-based digital library application that allows users to search for books from a vast collection using the Open Library API.
The goal of this project is to build an interactive and user-friendly interface where users can explore books, view details, and track their reading progress.

---

## 🎯 Purpose

This project is being developed as part of a graded assignment to demonstrate:

* API integration using **JavaScript (`fetch`)**
* Use of array methods like **`map`**, **`filter`**, and **`sort`**
* UI/UX design skills
* Writing clean and structured code

---

## 🌐 API Used

**Open Library API**

API Documentation:
https://openlibrary.org/developers/api

This API provides access to millions of books including:

* Book titles
* Authors
* Cover images
* Publish years

---

## ✨ Planned Features

### 🔍 Search Functionality

* Search books by **title** or **author**
* Display results dynamically on the page

### 🖼️ Book Display

* Show **book cover, title, author, and publish year**
* Display books in a **grid layout (bookshelf style)**

### 📊 Sorting

* Sort books based on **publish year**
* Options for **ascending / descending**

### ⚠️ Empty State Handling

* Show a message when **no results are found**

### 🎯 Reading Goal Tracker (Bonus Feature)

* Button: **"I've read this"**
* Clicking the button increments a **reading counter**
